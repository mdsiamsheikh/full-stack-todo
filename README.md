# Full-Stack-Todo

Developing a Full-Stack-Todo web app using Vue.js and Nuxt.js,and MongoDB postAPI, getAPI, updatAPI, deleatAPI, complete with secure login for Google and logout functionality through Google Firebase! Your use of CSS3 grid, flexbox, transform, and BEM convention for optimal UI design is commendable and contributes to the visually appealing and responsive experience of the app on various screen sizes.


Live version of this site: https://full-stack-todo1.netlify.app
